Assignment Ideas

  • Add Basic Controls (play/pause button)
    1. Create page elements for play and pause and make them interactive with event listeners.
    2. The event handler callback functions can be set to the existing play and pause methods in the SlideShow object.
  • Add configuration object as second parameter to SlideShow constructor, which will allow the user to configure the slide change delay.
    1. Add a second argument to the SlideShow( ) constructor function and assign it to a property in the constructor.The argument can be set in seconds and multiplied for the millisecond scale required by the setInterval function.
    2. Change the second parameter of the invocation of setInterval in the play method.
    3. Add the argument to the instantiation call of the SlideShow object to set the delay.
  • Add some more advanced controls: Forward/Backward buttons, clickable thumbnails, etc.
    1. Create page elements for forward and backward and make them interactive with event listeners.
    2. The forward event listener can be set to invoke the cycle( ) method as its callback.
    3. Declare a new recycle( ) function on the SlideShow prototype that does the reverse of cycle. That will be the callback for the Backward element.
  • Implement some form of pre-caching for the images.
    1. Declare an array in the SlideShow constructor.
    2. Create a new method on the SlideShow prototype called cache. The method should cycle through each of elements in the pictures object, create a new Image object in the DOM and load the source file and metadata.
    3. Refactor the render( ) method to accomodate that the elements are now available in the DOM to be added to the document.
    4. Invoke the cache( ) method - either on the page or in the constructor.
